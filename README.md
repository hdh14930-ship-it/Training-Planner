# Training Planner

Автор: Ермолаев Егор

## Описание

Это приложение позволяет планировать тренировки, добавлять их, фильтровать по дате и типу, сохранять и загружать данные с помощью JSON. Интерфейс реализован с помощью Python и библиотеки tkinter.

## Использование

1. Запустите программу.
2. Заполните поля «Дата», «Тип тренировки», «Длительность».
3. Нажмите «Добавить тренировку» — запись появится в таблице.
4. Для фильтрации выберите параметры и нажмите «Применить фильтр».

## Требования

- Python 3.x
- Библиотека tkinter (обычно входит по умолчанию)
- Библиотека json (встроенная)
